CSCE-145 — Java Concepts Review

By Khori Peterson

Key Topics We Covered

Accessors (Getters)

Accessors return private variable values from a class. Example:
public String getName() { return name; }

Mutators (Setters)

Mutators safely change private variable values. Example:
public void setAge(int a) { age = a; }

Encapsulation

Keeping variables private and controlling access with getters and setters.

toString() Method

Allows your object to print in a readable format.
public String toString() { return "Name: " + name; }

Mini Review Summary

  • Use getters to return private fields.
  • Use setters to safely modify data.
  • Encapsulation keeps your class secure.
  • toString() is crucial for printing objects nicely.
